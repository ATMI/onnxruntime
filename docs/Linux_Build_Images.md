This document describes the docker base images we use for building ONNX Runtime's official packages.

We use the following base images:


|             | Nuget             | Python                  |
|-------------|-------------------|-------------------------|
| x64 CPU     | Standard ubi8     | cpu_x64_ubi8_gcc12      |
| ARM64 CPU   | Standard almalinux    | cpu_aarch64_ubi8_gcc12      |
| x64 CUDA 12 | cuda12_2_x64_ubi8_gcc12_dotnet     | cuda12_2_x64_ubi8_gcc12      |
| x64 CUDA 11 | cuda11_x64_almalinux8_gcc11_dotnet     | cuda11_x64_almalinux8_gcc11      |
